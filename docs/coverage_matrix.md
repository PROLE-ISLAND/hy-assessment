## 空白セルの根拠

### 「—」の理由

| セル | 理由 |
|------|------|
| 候補者 × 認証成功 | 候補者はメールリンクでアクセス。認証フローなし |
| 候補者 × 候補者登録 | 候補者自身は登録しない。管理者が登録 |
| 候補者 × 検査リンク発行 | 管理者の操作。候補者は受信側 |
| 候補者 × AI分析生成 | システムの自動処理。候補者は関与しない |
| 候補者 × 分析結果閲覧 | 管理者専用機能。候補者は別途レポート閲覧 |
| 候補者 × 候補者比較 | 管理者専用機能 |
| 候補者 × レポート共有 | 管理者の操作。候補者は受信側 |
| 候補者 × テンプレート管理 | 管理者専用機能 |
| 候補者 × プロンプト管理 | 管理者専用機能 |
| 候補者 × 検査一覧閲覧 | 管理者専用機能 |
| 候補者 × ダッシュボード閲覧 | 管理者専用機能 |
| システム × その他全て | システムはAI分析のみ担当 |
| 管理者 × 回答データ保存 | 候補者の操作。管理者は閲覧のみ |
| 管理者 × 検査完了 | 候補者の操作。管理者は閲覧のみ |
| 管理者 × 候補者レポート閲覧 | 候補者専用機能（/report/[token]） |
| 管理者 × レポートリンク再取得 | 候補者のセルフサービス機能 |

### 「🟡」の理由

| セル | 理由 | テストレベル |
|------|------|-------------|
| 検査リンク発行 | 候補者登録と同一フローで統合テスト | 統合済み（UC-HY-ADMIN-CANDIDATE-WEB） |

| 候補者比較 | 付加価値機能。個別閲覧で代替可 | Silver E2E |
| テンプレート管理 | 初期設定のみ。頻度低 | Silver E2E |
| プロンプト管理 | 初期設定のみ。頻度低 | Silver E2E |
| 検査一覧閲覧 | 監視機能。分析閲覧で主要検証済み | Silver E2E |
| ダッシュボード閲覧 | 監視機能。分析閲覧で主要検証済み | Silver E2E |
| レポートリンク再取得 | セルフサービス。レポート閲覧で代替可 | Silver E2E |

## レビューチェックリスト

### 網羅性確認

- [x] **Roleが漏れてない？**
  - 管理者（ADMIN）: 採用担当者
  - 候補者（CAND）: 検査受験者
  - システム（SYS）: バックグラウンド処理
  - → オペレーター等は現行なし。将来追加時にUniverse更新

- [x] **Outcomeが漏れてない？**
  - 価値確定イベント**15個**を洗い出し（旧10個から拡充）
  - 「検査実施」「分析生成」「結果提供」「共有」の価値チェーン網羅
  - **候補者への価値提供**（レポート閲覧）を追加 ★重要

- [x] **空白セルに理由がある？**
  - 全「—」セルに根拠あり（上記表参照）
  - 全「🟡」セルにテストレベル明記

- [x] **Gold採用/却下の線引きが妥当？**
  - スコア13以上（20点満点）をGold採用
  - 事業継続に直接影響するユースケースのみ
  - **UC-HY-CAND-REPORT-WEB**は候補者への価値提供完結点として特別採用

### システム要件との整合性確認 ★NEW

- [x] **FEATURES.md記載機能との対応**
